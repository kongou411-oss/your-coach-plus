## 前提条件

### 価格設定
- **ベーシックプラン**: 740円/月
- **プレミアムプラン**: 980円/月
- **平均単価（ARPU）**: 860円/月（ベーシック60%、プレミアム40%の想定）

### 固定費
- **初年度（1-12ヶ月）**: 20万円/月
- **2年目（13-24ヶ月）**: 25万円/月（人件費追加想定）

## 4. バリュエーション試算（3年後）

### 前提条件
- **ARR（年間経常収益）**: 3年目の年間売上を使用
- **SaaS倍率**: ARR × 5-7倍（成長率・利益率を考慮）

### シナリオ別企業価値

| シナリオ | 3年後ARR | 倍率範囲 | 企業価値（下限） | 企業価値（上限） |
|---------|---------|---------|----------------|----------------|
| **保守的** | 347万円 | 3-4倍 | **1,041万円** | **1,388万円** |
| **標準** | 1,712万円 | 5-6倍 | **8,560万円** | **1億271万円** |
| **楽観的** | 10,392万円 | 6-8倍 | **6億2,352万円** | **8億3,136万円** |

### 補足説明

#### 保守的シナリオ
- 成長が鈍化し累積赤字のため、倍率は低く設定（3-4倍）
- 企業価値は1,000万円台にとどまる
- 追加投資なしでは事業継続が困難

#### 標準シナリオ
- 健全な成長と黒字化を達成、SaaS標準倍率5-6倍を適用
- 企業価値は8,560万円〜1億円規模
- シリーズAの資金調達が視野に入る水準

#### 楽観的シナリオ
- 急成長と高い利益率により、高倍率6-8倍を適用
- 企業価値は6億円超の可能性
- 大型資金調達やエグジット（M&A/IPO）が現実的に

## 5. シナリオ比較サマリー

### 3年後の姿

| 指標 | 保守的 | 標準 | 楽観的 |
|------|--------|------|--------|
| **会員数** | 398人 | 1,824人 | 12,069人 |
| **年間売上** | 347万円 | 1,712万円 | 10,392万円 |
| **累積利益** | -385万円 | +1,280万円 | +11,351万円 |
| **企業価値** | 1,041-1,388万円 | 8,560万円-1億円 | 6.2-8.3億円 |
| **黒字化** | 未達 | 1年3ヶ月 | 即時 |

### 重要な示唆

1. **標準シナリオの実現可能性が高い**
   - 月次成長率5%、解約率4%は業界標準の範囲内
   - 1年3ヶ月での黒字化は現実的
   - 2年目から安定収益を生み出す

2. **初期会員獲得が成否を分ける**
   - 初月200人の獲得が標準シナリオの前提
   - ローンチ時のマーケティングが極めて重要
   - B2B2C連携が初期会員獲得の鍵

3. **解約率のコントロールが収益性を左右**
   - 5%→4%への1%改善で、3年後の利益が大きく変わる
   - プロダクトの品質とユーザー体験が最優先

4. **楽観的シナリオは戦略次第で実現可能**
   - インフルエンサー連携、バイラル機能の実装
   - B2B2C大型契約の獲得
   - メディア露出による認知拡大

## 6. 推奨アクション

### フェーズ1（0-6ヶ月）：ローンチと初期獲得
- **目標**: 200人の初期会員獲得（標準シナリオ）
- **施策**:
  - β版での徹底的なフィードバック収集
  - ローンチキャンペーン（初月無料など）
  - インフルエンサー連携2-3件
  - B2B2Cパートナー1-2社獲得

### フェーズ2（7-18ヶ月）：成長と黒字化
- **目標**: 月次成長率5%維持、解約率4%以下
- **施策**:
  - プロダクト改善（解約理由分析）
  - コンテンツマーケティング強化
  - リファラルプログラム導入
  - B2B2C拡大（5社以上）

### フェーズ3（19-36ヶ月）：スケールアップ
- **目標**: 会員数1,500人突破、年間売上1,500万円
- **施策**:
  - 有料広告の本格展開
  - エンタープライズ向けプラン追加
  - 海外展開の検討
  - 資金調達の準備（シリーズA）

## 計算ロジック詳細

### 会員数の計算式

```
翌月会員数 = 当月会員数 × (1 - 解約率) + 新規会員数
新規会員数 = 当月会員数 × 月次成長率
```

### 売上の計算式

```
月次売上 = 会員数 × ARPU（860円）
年間売上 = 各月の月次売上の合計
```

### 費用の計算式

```
月次費用 = 固定費 + 決済手数料 + インフラコスト + CAC
決済手数料 = 月次売上 × 3.6%
インフラコスト = 会員数 × 50円
CAC = 新規会員数 × 2,000円
```

### 利益の計算式

```
月次利益 = 月次売上 - 月次費用
累積利益 = 過去すべての月次利益の合計
```

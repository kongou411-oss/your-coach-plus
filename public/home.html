FEATURE 01

最速1分で記録から分析まで完了

テンプレート・ルーティン機能で普段の食事や運動をワンタップ入力。 それでいて、食事・運動・コンディションの3要素を組み合わせた本格的な多要素分析が可能。

  • ✓ PFC・ビタミン・ミネラル 40項目以上を自動計算
  • ✓ 睡眠・ストレス・腸内環境などコンディションも考慮
  • ✓ AIが総合スコアで毎日の状態を可視化
FEATURE 02

学習→実践→改善がアプリ内で完結

PGBASE（情報データベース）で栄養・トレーニングの基礎を学び、 そのまま実践。記録データをAIが分析し、改善点をフィードバック。

  • ✓ 科学的根拠に基づいた教科書コンテンツ
  • ✓ 学んだ内容をその場で実践・記録
  • ✓ AIが個人に最適化されたアドバイスを提供
FEATURE 03

ボディメイクを「プロジェクト」として共有

COMY（コミュニティ）で自分の体づくりをプロジェクト化。 同じ目標を持つ仲間と進捗を共有し、モチベーションを維持。

  • ✓ 目標・期間を設定してプロジェクト作成
  • ✓ 日々の記録を自動で投稿・共有
  • ✓ いいね・コメントで仲間と励まし合い

⚖️

なぜ「LBM」を中心にしたのか

LBM（除脂肪体重）こそがボディメイクの中心。筋肉量を維持・増加させながら体脂肪を減らす。これが本当の「体づくり」です。

🎯

他アプリとの違い

「簡単入力」と「詳細分析」の両立。テンプレート・ルーティン機能で最速1分で記録から分析まで完了。それでいて40項目以上の栄養素を追跡。
